1. About the App

Hours — Time Zone Converter is a utility application for time zone reference. The App allows users to create and manage a list of cities and compare their time zones.

Key features include:

  • viewing current time in multiple cities;
  • adjusting a shared time offset using a rotary dial interface;
  • adding cities from a built-in database;
  • searching for cities using Apple MapKit services;
  • optionally using the device’s location to suggest the current city;
  • renaming, reordering, and deleting cities;
  • switching time format (12-hour / 24-hour); and
  • switching appearance mode (system / light / dark).
